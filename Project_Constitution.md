# Mohammed Hasan Raza OS

## Project Constitution v1.0

---

# Vision

Mohammed Hasan Raza OS is not a portfolio website.

It is a Game Developer Operating System.

The platform exists to showcase:

* Game Development
* SDK Engineering
* Technical Thinking
* Engineering Case Studies
* Professional Growth
* Future Ambitions

The website should communicate technical capability, professionalism, and long-term potential within the game industry.

---

# Mission

Create a premium recruiter-focused platform that documents:

* Games built
* Systems engineered
* Problems solved
* Lessons learned
* Career progression

The site must remain valuable for the next 3–5 years without requiring a redesign.

# Design Philosophy

The website should feel like:

AAA Studio Website
+
Developer Command Center

Balance:

70% Technical
30% Creative

Avoid:

* Cyberpunk clichés
* RGB aesthetics
* Excessive glassmorphism
* Particle backgrounds
* Hacker themes
* Generic portfolio layouts

---

# Design Principles

1. Content First

Content should always be more important than visual effects.

2. Purposeful Motion

Animation must communicate information.

Never animate for decoration alone.

3. Recruiter Efficiency

Visitors should understand:

* Who Mohammed is
* What he builds
* Why he is valuable

Within 10 seconds.

4. Scalability

Every section must support future content growth.
